## AI Interactions

In this project, I used an AI assistant to help debug and implement dark mode in the To-Do app.

- **Prompt:** "How should I implement Dark Mode in my application?"  
  **Response:** The AI assistant identified which UI elements needed styling and provided a beginner-friendly CSS file (`dark.css`) along with guidance on how to connect it to the app.

- **Prompt:** "Improve this code for readability and organization"  
  **Response:** The AI assistant suggested improvements to the organization of the `TaskManager` and `MainUIController` code to make it cleaner and easier to understand.

- **Prompt:** "Why am I getting this error in my Java code?"  
  **Response:** The AI assistant explained what was wrong with the code and guided me on how to fix the issues.
